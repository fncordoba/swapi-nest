# SWAPI-Test

1. Clonar proyecto
2. ```yarn install```
3. Clonar el archivo ```.env.template```
4. renombrarlo a ```.env```
5. Levantar la base de datos
```
docker-compose up -d
```

6. Levantar: ```yarn start:dev```

7. Ejecutar SEED en caso de querer usuarios de prueba
```
http://localhost:3000/api/seed
```

8. Ejecutar pruebas
```
yarn test
```
